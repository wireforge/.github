<h1 align="center">wireforge</h1>

<p align="center"><strong>Send. Shape. Ship.</strong></p>

<p align="center">A lightweight, open-source API client. Over the wire.</p>

---

**wireforge** is a fast, keyboard-first API client built as a real native app — not a browser tab. Requests run in Rust, not the webview, so there's no CORS wall between you and your API. Collections live as files in your repo, so the whole team forges requests the same way they ship code.

### Why wireforge

- **Lightweight by design** — Tauri + Rust + Svelte. System webview, no bundled Chromium.
- **Git-native collaboration** — collections are plain files in your repo. Branch, review, merge — like code.
- **GraphQL, done right** — a schema-aware editor with introspection-powered autocomplete.
- **Fast on big responses** — streaming + virtualized rendering where other clients choke.
- **AI-native** — designed so coding agents can drive it.

### Status

Early, and built in the open. Pre-`1.0` — APIs and storage formats may still move. Stars, issues, and PRs welcome.
